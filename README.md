### [6.프로젝트 진행 계획서]

- React,react를 활용한 공공 데이터 정보 제공 웹 서비스 개발
전체적인 프로젝트 진행에 있어서 코드 관리 및 이슈 관리는 Slack, Github 사용하였습니다. 또한, 전체적인 소스코드의 충돌을 줄이기위하여 Github PullRequest를 주 기반으로하여 프로젝트 통합을 진행하였습니다. 프로젝트의 목표는 공공 데이터센터의 API를 활용하여 데이터 가공하여 데이터베이스 테이블을 구축 한 프로젝트를 진행 할것이며, Javascript 기반의 React 를 활용한 프론트 개발과 함께 Non-blocking I/O와 단일 스레드 이벤트 루프를 통한 높은 처리 성능을 가진 Node.js기반의 서버를 구축할 것입니다. 또한, 데이터베이스는 대표적인 Nosql중 하나인 MongoDB를 활용하여 전체적인 스키마 설계 및 다이어그램 작성으로 프로젝트를 진행하려고 합니다. 주 1-2회씩 필요에 따라 프로젝트 개발 및 회의를 진행할것이며, 팀장,팀원들간의 의견 조율을 통하여 프로젝트 개발 예정입니다.
